• Basic layer runs on http://localhost:8000.
  • Web layer runs on http://localhost:8001.

Some links to illustrate templates and static resources in layers.

Basic.
  • http://localhost:8000/foo/.
  • http://localhost:8000/bar/.
  • http://localhost:8000/admin/.
  • http://localhost:8000/static/example/foo.css.
  • http://localhost:8000/static/example/bar.css.
Web. All bar URLs are custom for this layer, as well as /admin.
  • http://localhost:8001/foo/.
  • http://localhost:8001/bar/.
  • http://localhost:8001/admin/.
  • http://localhost:8001/static/example/foo.css.
  • http://localhost:8001/static/example/bar.css.
